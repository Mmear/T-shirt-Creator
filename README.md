# T-shirtCreator

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 目录结构
- public 展示文件
- src 源文件
  - assets 静态文件
  - components vue组件
  - routes  路由
  - store vuex状态管理
    - modules 各个状态模块
  - test 数据模拟 http测试
  - utils 全局功能
  - view 各功能页面
  - App.vue 主界面
  - main.js 主要脚本文件(引入各类资源，创建Vue实例)
- babel.config.js babel设置
- vue.config.js 项目脚手架设置
